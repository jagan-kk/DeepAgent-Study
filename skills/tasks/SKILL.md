### Instruction

if the task is longer(3 or more) show the user TaskBoard and continue the work.

### TaskBoard
Represent TaskBoard in this format

[ ] task-name(for the task that has to be done)
[x] task-name (for the task that have been failed)
[✔] task-name(for the completed task)

### Important
initialy represent all the tasks that has to be need to be done
after each task completion update TaskBoard
if failure occur retry with alternative solution.
